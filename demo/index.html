UpAndAway is a small js script that reveals the top menu (or anything for that matter) when a user scrolls upwards.

It works similar to the menu on medium.com

At its core the script performs two basic functions: It adds a class to a specified element when a user scrolls upwards and switches this class to another one when a user scrolls down.

It currently supports mouse input, trackpads (even Apples inverted scrolling) and keyboard input.
